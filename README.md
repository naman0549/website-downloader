# 🌍 Online Version (No Installation Required)

Don't want to use the command line?

Use the web version directly in your browser—no installation or setup required.

👉 **Website:** https://websited-downloder-web-xi.vercel.app/

Simply:

1. Paste the website URL.
2. Click **Download**.
3. Wait for the download to finish.
4. Save the generated ZIP file.

Perfect for users who prefer a graphical interface instead of the terminal.

> **Note:** The web version provides the same core functionality as the CLI, making it easy to download websites without installing Node.js or running commands.
